
# Baixar cursos da alura

# Como utilizar

## Antes de qualquer coisa, renomeiei o arquivo ".env.example" para ".env" e informe suas credenciais

1 - No arquivo dtConfig.js, informe a url do curso desejado, conforme link da alura.

2 - npm install

3 - node index.js
